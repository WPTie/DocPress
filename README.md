# README #

Create your product's docs with a WordPress theme

### DocPress ###

* Document your product
* Version 1.0.0

### How do I get set up? ###

*  Install the theme 
* Install required plugin from TGM
* Add Categories for top level menus
* Add posts in the categories for sub-level menus
